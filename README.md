# Jack's Roofing LLC - Website

Professional website for Jack's Roofing LLC, a roofing service company.

## About

This is a simple, responsive landing page for Jack's Roofing LLC featuring:
- Company information and services
- Contact form for quote requests
- Mobile-responsive design
- Clean, professional styling

## Structure

- `index.html` - Main landing page
- `styles.css` - Stylesheet
- `README.md` - This file

## Local Development

### Option 1: Using npm (Recommended)

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

Or use:
```bash
npm run dev
```

The site will open automatically in your browser at `http://localhost:3000`

### Option 2: Simple Method

Simply open `index.html` in a web browser to view the site.

## Technologies

- HTML5
- CSS3
- Responsive design (mobile-friendly)
